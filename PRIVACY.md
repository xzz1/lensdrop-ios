# LensDrop Privacy Policy

Effective date: May 23, 2026

LensDrop is designed for offline optical file transfer. A computer sender displays a file as animated color icon matrix barcodes, and LensDrop reconstructs the file locally by scanning those images through the camera.

## Personal Information

LensDrop does not collect, sell, or upload personal information.

## Camera

LensDrop requests camera access only to scan cimbar images and reconstruct files on your device. Camera frames are not transmitted by LensDrop to any server.

## Received Files

Completed received files are made available for you to export. A file is written only to the location you select through the system file interface.

## Preferences

Your language selection is stored locally on your device.

## External Website

When you choose **Get Latest Sender from libcimbar**, LensDrop opens an external GitHub download page. External websites are governed by their own privacy practices.

## Offline Sender

LensDrop includes an offline sender HTML file that you may export and open locally in a web browser. LensDrop itself does not run that page or receive the file you select in the sender.

## Changes

If LensDrop later adds network services, analytics, accounts, subscriptions, or other data-handling features, this policy will be updated before those features are released.

# LensDrop 隐私政策

生效日期：2026 年 5 月 23 日

LensDrop 专为离线光学文件传输而设计。电脑端发送页将文件显示为动态彩色图标矩阵条形码，LensDrop 通过相机扫描这些图像并在设备本地恢复文件。

## 个人信息

LensDrop 不收集、出售或上传个人信息。

## 相机

LensDrop 仅为在你的设备上扫描 cimbar 图像和恢复文件而请求相机权限。LensDrop 不会将相机画面发送到任何服务器。

## 接收的文件

完整接收的文件可供你主动导出；文件只会写入你通过系统文件界面选择的位置。

## 偏好设置

你的语言选择仅保存在设备本地。

## 外部网站

当你选择 **从 libcimbar 获取最新发送端** 时，LensDrop 会打开外部 GitHub 下载页面。外部网站适用其自身的隐私规则。

## 离线发送端

LensDrop 包含可导出的离线发送端 HTML 文件，你可以在浏览器中本地打开使用。LensDrop 本身不会运行该页面或接收发送端所选择的文件。

## 政策更新

如果 LensDrop 未来增加网络服务、数据分析、账户、订阅或其他数据处理功能，将在这些功能发布前更新本政策。
